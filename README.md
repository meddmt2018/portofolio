# 🌌 Farrell Ahmed — Web Portofolio

Website portofolio pribadi bertema luar angkasa, dibangun tanpa framework berat.

🔗 **Live:** [ahmed-dimitrie.netlify.app](https://ahmed-dimitrie.netlify.app)

---

## ✨ Fitur

- Animasi background luar angkasa (bintang, meteor, aurora, nebula)
- Hero section dengan typing animation dan blackhole effect
- Filter proyek & sertifikat berdasarkan kategori
- Render thumbnail sertifikat PDF otomatis via PDF.js
- Cover proyek dengan badge GitHub overlay
- Data tersimpan di localStorage
- Responsif di desktop & mobile
